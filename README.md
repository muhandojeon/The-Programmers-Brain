# 프로그래머의 뇌

인지과학에 기반을 둔 각종 방법론으로 개발자가 새로운 언어나 프레임워크를 빠르게 배워 생산성을 향상하도록 돕는다. 코드를 더 잘 이해하는 것에서 나아가, 버그 방지, 명확한 작명법, 리팩터링, 대규모 코드베이스 개선에 이르기까지 과학적으로 입증된 기법을 제시한다.

## 진행 방식 🌌

* 스터디 세션 전에 주차에 해당하는 장을 읽고 정리한 내용을 PR로 올려요
* 스터디 세션에서 각자 아래 주제에 대해 공유해요
  + 중요하다고 생각한 내용
  + 경험적인 내용
  + 이해가 되지 않는 내용
* 각 장마다 정해진 서기가 공유 내용을 기록하고 저장소에 올려요
