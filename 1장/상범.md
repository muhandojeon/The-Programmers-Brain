# 챕터1 코딩 중 겪는 혼란에 대한 이해

우리가 여러 종류의 혼란을 겪을 때 다음과 같은 서로 다른 종류의 인지 과정이 관련된다
- 지식의 부족 => LTM의 문제
  - LTM: 하드 드라이브 
- 정보의 부족 => STM의 문제
  - STM: 캐시나 메인 메모리 
- 처리 능력의 부족 => 작업 기억 공간의 문제
  - 작업 기억 공간: CPU
